## Documentation Format

Each action card document includes:

- Basic information (cost, type, rarity)
- Effect description
- Strategic usage
- Upgrade paths
- Flavor text
- Visual references
- Balance data

## Contributing

To add or modify action cards:

1. Create a new markdown file in the appropriate faction directory
2. Follow the standard template
3. Update this README with links to new cards
4. Submit a pull request for review
